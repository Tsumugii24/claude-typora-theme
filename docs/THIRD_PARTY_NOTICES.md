# Third-Party Notices

This repository bundles theme assets and font files for offline Typora usage.

## Original Theme

This project is based on the Claude Typora theme by blaxisomu:

- Repository: https://github.com/blaxisomu/CLAUDE-Typora
- Upstream license metadata: no license was reported by GitHub at the time this package was prepared.

The local version keeps selected user-facing preferences and integrates later upstream styling details.
